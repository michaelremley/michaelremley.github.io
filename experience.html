The Lorenz system is a famous example, and my faculty sponsor, Diana Dabby, for this research knew him personally. Summary
Engineering Systems Analysis, or ESA for short, is a class that follows QEA to teach important systems-related math. We lead students into the mysterious s-domain and z-domain worlds and teach them to use them via a project where they balance an inverted pendulum robot named Rocky. This course is fairly new as of writing, so I have spent time revising assignments and creating solution keys for assessment.

Summary
Quantitative Engineering Analysis, or QEA for short, is a 8-credit hour class that combines physics, algebra, and engineering into a series of team-based projects. The eigenfaces project teaches linear algebra up to eigenvectors using a primitive facial recognition program and focuses on the ethics of technology. I provided some assets to make the remote version of the robotics module, which ran in a simulator in Spring 2020, more engaging and friendly to the students. One of these is pictured to the right.
